현재 프로젝트의 개발 Phase 진행 상황을 보고합니다.

1. `docs/ROADMAP.md`를 읽어서 전체 Phase 목록과 태스크를 확인합니다.
2. 각 태스크의 완료 여부를 실제 코드 존재 여부로 판단합니다:
   - 해당 파일/모듈이 존재하고 기본 기능이 구현되어 있으면 완료
   - 파일은 있지만 스텁/TODO만 있으면 진행 중
   - 파일이 없으면 미착수
3. Phase별 진행률(%)과 다음에 해야 할 작업을 요약합니다.

출력 형식:
```
Phase 1: Core Prototype [3/6 완료 - 50%]
  ✅ 프로젝트 셋업
  ✅ 지형 생성
  🔄 포탄 물리 (진행 중)
  ⬜ 지형 파괴
  ⬜ 턴 시스템
  ⬜ 기본 UI

다음 작업: 포탄 물리 시뮬레이션 완료
```
