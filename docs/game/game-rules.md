# Scorched Modern - Game Rules & Balance

> 이 문서는 GDD에서 다루지 않는 세부 게임 규칙, 밸런스 수치, 엣지 케이스를 정의합니다.
> 각 Phase 구현 시 발견되는 규칙을 지속적으로 추가합니다.

---

## 1. Turn Rules

### 1.1 Turn Order
- 라운드 시작 시 모든 플레이어에게 랜덤 턴 순서 배정
- 다음 라운드에서는 이전 라운드 역순 (공정성)
- 죽은 플레이어는 턴 스킵

### 1.2 Turn Timer
- 제한 시간: 30초
- 타임아웃 시: 기본 포탄을 랜덤 각도(30~150도), 랜덤 파워(30~70%)로 자동 발사
- 이동 + 발사는 같은 턴 내에서 수행 (이동 먼저, 발사 후)

### 1.3 Turn Actions
한 턴에 가능한 행동:
1. 이동 (선택, fuel 소비)
2. 각도/파워 조절
3. 무기 선택
4. 발사 (1회)

발사 없이 턴을 넘길 수 있음 (Skip 버튼).

## 2. Damage & HP

### 2.1 Base Damage
데미지는 폭발 중심으로부터의 거리에 반비례:

```
actual_damage = base_damage * (1 - distance / radius)
```

- 폭발 중심에 있으면 100% 데미지
- 반경 경계에 있으면 0% 데미지
- 반경 밖이면 데미지 없음

### 2.2 Self-Damage
- 자신의 포탄에 의한 데미지 적용 (자폭 가능)
- 자폭 시 골드 보상 없음

### 2.3 Kill Rules
- HP가 0 이하가 되면 즉시 사망
- 동일 폭발로 여러 탱크 사망 시 모두 킬로 인정 (멀티킬)
- 마지막 생존자가 자폭하면 무승부 (해당 라운드 승자 없음)

## 4. Economy Balance

### 4.1 Gold Rewards
| Event | Gold |
|-------|------|
| 시작 자금 | 500G |
| 적 적중 | `damage * 2` |
| 킬 | +200G |
| 라운드 생존 | +100G |
| 라운드 승리 | +300G |

### 4.2 Shop Rules
- 모든 무기는 1개 단위로 구매
- 구매 취소 가능 (Shop Phase 종료 전)
- 미사용 무기는 다음 라운드로 이월

## 5. Wind

- 범위: -200 ~ +200
- 양수: 오른쪽으로 밀림, 음수: 왼쪽으로 밀림
- 턴마다 최대 +-50 변화 (급격한 변화 방지)
- 바람은 수평 방향에만 영향 (수직 영향 없음)

---

## 6. Terrain Rules

### 6.1 Tank Placement
- 라운드 시작 시 지형 위에 균등 간격 배치
- 최소 간격: 탱크 너비의 5배 (밀집 방지)
- 경사가 너무 급한 곳(45도 이상)에는 배치하지 않음

### 6.2 Tank Gravity
- 지형 파괴로 탱크 아래가 비면 탱크도 추락
- 추락 데미지: `fall_height / 10` (최대 30)
- 추락 후 지형 표면에 착지

### 6.3 Buried State
- 탱크 상단 50% 이상이 지형에 묻히면 Buried
- Buried 상태: 이동 불가, 발사는 가능
- 다음 턴 시작 시 Buried 해제 없음 (자연 해제 없음, 주변 폭발로만 해제)

## 7. Edge Cases

### 7.1 Last Player Standing
- 자기 차례에 자폭하여 마지막 생존자가 없어지면 → 무승부
- 다른 플레이어의 포탄으로 마지막 2인이 동시 사망 → 발사자가 먼저 사망 판정, 상대가 승리

### 7.2 Disconnection (Multiplayer)
- 연결 끊김 시 해당 플레이어 탱크는 AI(Easy)로 대체
- 30초 내 재접속 시 제어권 복구
- 재접속 불가 시 AI가 라운드 끝까지 대체

### 7.3 Map Boundary
- 포탄이 좌/우 화면 밖으로 나가면 소멸 (wrap 없음)
- 포탄이 상단을 넘어가면 계속 비행 (중력으로 돌아옴)
- 포탄이 하단 경계 아래로 내려가면 소멸

## 8. Phase 1 Scope Rules

Phase 1 프로토타입에서 적용할 최소 규칙:
- Standard Shell만 구현
- 2인 로컬 플레이 (네트워크 없음)
- 바람 없음 (0 고정)
- 경제 시스템 없음
- 턴 타이머 없음 (수동 턴 전환)
- 자폭 데미지 적용
- 추락 데미지 미적용
